### constants.py

Centralised names for all topics and services. Import instead of hardcoding strings:


### Topics

#### /learning/status

Publishes robot status once per second.

**Type:** `std_msgs/msg/String`
**Format:** `battery=<value>; mode=<value>; speed=<value>`

| Field | Behaviour |
|---|---|
| `battery` | Decreases by 0.5‚Äì1.5 per second, resets to 100 at 0 |
| `mode` | Alternates between `manual` and `auto` |
| `speed` | Random value 0.0‚Äì1.0 |

```bash
# Run
ros2 run learning topics_status_pub
ros2 run learning topics_status_sub

# Inspect
ros2 topic echo /learning/status
ros2 topic info /learning/status
```

---

#### /learning/command

Publishes a robot command every 2 seconds, cycling through the full sequence.

**Type:** `std_msgs/msg/String`
**Commands:** `forward` ‚Üí `backward` ‚Üí `left` ‚Üí `right` ‚Üí `stop` ‚Üí repeat

```bash
# Run
ros2 run learning topics_command_pub
ros2 run learning topics_command_sub

# Inspect
ros2 topic echo /learning/command

# Send manually
ros2 topic pub /learning/command std_msgs/msg/String "{data: 'forward'}"
ros2 topic pub --once /learning/command std_msgs/msg/String "{data: 'forward'}"
```
